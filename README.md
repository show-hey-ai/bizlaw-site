# Bizlaw Site - ビジネス法務検定3級 毎日学習サイト

毎日新しいビジネス法務検定3級の問題と合格体験談、励ましの名言を自動生成・公開するWebサイトです。

## 🌟 概要

このサイトは、ビジネス法務検定3級の学習者向けに、以下のコンテンツを毎日自動更新で提供します：

- 📝 **10問の○×問題**（解説付き）
- ✨ **日替わりの名言**（偉人の名言で学習のモチベーションアップ）
- 💪 **合格体験談**（励みになる体験談を毎日更新）
- 📚 **過去の問題アーカイブ**（蓄積されたすべての過去問題にアクセス可能）
- 🤖 **AI学習サポートチャット**（Google Gemini API使用）

## 🛠️ セットアップ

### 1. リポジトリのクローン

```bash
git clone https://github.com/show-hey-ai/bizlaw-site.git
cd bizlaw-site
```

### 2. Python環境のセットアップ

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

### 3. Google Gemini APIキーの設定

#### 方法1: 直接設定（推奨）
サイトを開いて「APIキーを設定」ボタンをクリックし、APIキーを入力

#### 方法2: コード内で設定
`index.html`内の以下の行にAPIキーを設定：
```javascript
let GEMINI_API_KEY = localStorage.getItem('gemini_api_key') || 'YOUR_API_KEY_HERE';
```

APIキーの取得：[Google AI Studio](https://makersuite.google.com/app/apikey)

### 4. サーバーの起動

#### Windows（推奨）
```bash
# 方法1: バッチファイルを使用
start.bat

# 方法2: 手動で起動
python server.py
```

#### Mac/Linux
```bash
python3 -m http.server 8080
```

### 5. アクセス

ブラウザで以下のURLにアクセス：
```
http://localhost:8080
```

**注意**: ポート番号が8000から8080に変更されました。

## 🐛 トラブルシューティング

### よくある問題

1. **APIキーエラー**
   - サイト上の「APIキーを設定」ボタンから新しいキーを設定
   - または[Google AI Studio](https://makersuite.google.com/app/apikey)で新しいキーを取得
   - モデル名が`gemini-1.5-flash`になっていることを確認

2. **データが表示されない / 読み込みエラー**
   - **ローカルサーバーを使用**：`start.bat`をダブルクリック、または`python server.py`を実行
   - ブラウザで`http://localhost:8080`にアクセス（ポート8080に注意）
   - HTMLファイルを直接開いている場合は、サンプルデータのみ表示されます
   - `data/`フォルダに該当日付のJSONファイルがあるか確認

3. **AIチャットが動作しない**
   - APIキーが正しく設定されているか確認
   - インターネット接続を確認
   - APIの利用制限に達していないか確認
   - ブラウザのコンソールでエラーメッセージを確認

4. **過去の問題が表示されない**
   - ローカルサーバーを使用しているか確認（`http://localhost:8080`）
   - HTMLファイルを直接開いている場合は、サンプル問題が表示されます
   - 日付ボタンをクリックしているか確認
   - 同じ日付を再度クリックすると閉じる仕様

5. **ポート8080が使用中**
   - `server.py`の`PORT = 8080`を別の番号（例：8081）に変更
   - `start.bat`内のURLも同じポート番号に変更
   - ブラウザでアクセスする際も新しいポート番号を使用

6. **ローカルファイルモードの制限**
   - HTMLファイルを直接開いた場合、実際のデータファイルは読み込めません
   - サンプルデータのみ表示されます
   - 完全な機能にはローカルサーバーが必要です
